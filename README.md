# 📌 Overview

**Shopping Cart App** is a modern and fully responsive **React-based E-Commerce Shopping Cart** built using **React, Vite and JavaScript**.

The application allows users to browse products, search items, filter categories, manage shopping carts, update quantities, and view the total price in real time with a clean and modern shopping experience.

Designed with a premium responsive UI inspired by modern online shopping platforms.
